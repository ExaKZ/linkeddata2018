LinkedIn Career Trajectory Dataset (Description Only)

Overview

This repository provides a description of a LinkedIn-based career trajectory dataset collected in 2018 for academic research purposes.
Due to ethical, legal, and platform-related restrictions, the raw dataset is not publicly available.

The dataset was used to study:
	•	career progression patterns
	•	job transition dynamics
	•	industry mobility
	•	skills and role evolution

⸻

Data Collection
	•	Source: Public LinkedIn profiles (scraped in 2018)
	•	Ethics approval: Granted in 2018 by the relevant institutional ethics committee
	•	Access scope: Approved for academic research use only

⸻

Anonymisation

The dataset contains:
	•	❌ No names
	•	❌ No usernames
	•	❌ No profile URLs

All direct personal identifiers were removed prior to analysis.
However, the data may still contain indirect identifiers (e.g., job sequences, timelines), and therefore cannot be publicly released.

⸻

Dataset Structure (High-Level)

The dataset includes the following attributes (example):
	•	profile_id (pseudonymised)
	•	job_title
	•	industry
	•	company_size_category
	•	career_start_year
	•	job_start_year
	•	job_end_year
	•	employment_type
	•	skills_category

⚠️ Exact values, mappings, and raw records are not included in this repository.

⸻

Intended Use

The dataset is intended for:
	•	non-commercial academic research
	•	reproducibility verification
	•	methodological comparison

⸻
